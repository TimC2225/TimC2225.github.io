    • Exercise - HTML and CSS:

    For the first potion of assignment 2, we first started by watching the video on the right. Through this video, we learned how to set up a basic HTML-based website, specifically learning how to insert a title, headers, paragraphs, breaks, lines, and images.

    For the second portion of assignment 2, we continued by watching the video on the left. Through this video, we learned how to plan out a design for a website and use CSS to make the website more aesthetically pleasing and easy to navigate.

    In particular, we learned how to change the colour of backgrounds, text, and other containers, learned how to add padding and margins to different components of the website, learned how to use float to shift the various compenents of the website to the left or right of the screen, and learned how to change the dimensions of the components of the website among other functions.
