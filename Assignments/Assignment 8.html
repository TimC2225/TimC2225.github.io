    Assignment 5 - Tinkercad

    Task 1

    For Task 1, we were tasked to connect an LED to an arduino and write a program to blink the LED in a variety of patterns. I started by connecting the LED to pin 6, then wrote code that causes the led to change logic level every 1000ms then every 500ms before looping.

    Task 2

    For Task 2, we were tasked to connect 3 LEDs to an arduino and write a program to produce running lights, i.e. the 3 LEDs are lit in sequence. I started by connecting the LEDs to pin 6, 5 and 4, then wrote code that causes the leds to turn on then off in sequence before looping.

    Task 3

    For Task 3, we were tasked to connect a button to the circuit from Task 1, before writing a program to toggle the logic level of the LED when the button is pressed. I started by connecting the button to pin 7, before writing the program to toggle the logic level of the leds. A while loop at the end of the main loop was used to make sure that the logic level was only toggled once per press even if the button was held down.

    Task 4

    For Task 4, we were tasked to use the switch in task 3 to control the type of light-up and flashing of 3 LEDs connected similarly to Task 2, such that the leds all start in the OFF state, then each light up in sequence as the button is pressed, before all turning on after the 4th press, then reseting back to the first state. I started by connecting a switch like in Task 3 to the circuit from Task 2, before writing the code to turn on the appropriate leds after each button press, with the arduino keeping track of the number of presses using the "sw" and "sw_2" integer variables.
