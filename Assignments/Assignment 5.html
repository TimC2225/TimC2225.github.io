    1. For all of these exercises, we used Fusion 360 to create 3D models based on video guides and design specifications. In exercise 5, we made a toy brick by, firstly, using the sketch tool to make a 31.80mm by 15.80mm rectangle, which was then extruded by 9.60mm to form the basic shape of the brick. We then used the sketch tool to draw a circle on the top of the brick.

    2. Afterward, we extruded the circles and duplicated using the rectangular pattern tool to create the cylinders on the top of the brick.

    3. Then, we used the shell tool to hollow out the bottom of the brick. From the hollowed out area, we sketched the circles required to make the cylinders in the bottom of the brick, before extruding these circles and using the shell tool to hoollow out the cylinders.

    4. The top cylinders were then chamfered, resulting in the above 3D model.
