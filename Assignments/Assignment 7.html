    • Week 10 - Arduino I

    Exercise 13

    For exercise 13, we wired up a circuit using an arduino, led and resistor as per an image, before writing a program to blink an external LED. The LED was connected in series with a resistor to pin 8, with that pin being set as an output. Code was then written that causes the LED to change logic level every 1000ms.

    Exercise 14

    For exercise 14, we wired up a 7-segment display, power supply, switches and resistors, with the switches controlling each segment of the display, allowing the segments to display various numbers and letters.

    Exercise 15

    For exercise 15, we wired up a 7-segment display, resistors and an arduino, before writing a program to display the numbers 0 to 9 in sequence continuously. The program sets all the pins used as outputs, and lights up the appropriate segments of the display to display the each number in sequence before looping.

    Exercise 16

    For exercise 16, we wired up an LED and a resistor, and used the serial monitor to count the number of blinks of the led. The code written sets the led at pin 13 as an output and causes the led to toggle logic states every 500ms, with the integer variable "a" increasing and being printed in the serial monitor with every loop to count the number of blinks.
